# PowSignal / Freshie Finder

**URL:** freshie.ai
**Status:** In beta
**Type:** Consumer / travel / AI-native side project

## What It Is

PowSignal (also known as Freshie Finder) is an AI-powered travel platform that surfaces last-minute powder trips by combining real-time storm forecasting, resort conditions across 6,000+ destinations, and live travel pricing. The idea is simple: if a storm is hitting a mountain this weekend, you should know about it and be able to book a trip around it in one place.

## Why I Built It

Two reasons. First, it's a real problem I have — I snowboard and figuring out where powder is actually going to be is a multi-tab, multi-app process that takes way too long. Second, I wanted to build something end-to-end using AI-native development tools to see how far I could get and how fast.

## How It's Built

Built entirely with AI-native development tools: Claude Code, Cursor, and Lovable for the frontend. React for the UI, Supabase for the database and backend, Vercel for deployment. No traditional engineering team — just me and the tools.

The data layer pulls from real-time storm forecasting APIs, resort condition feeds, and live travel pricing sources and synthesizes them into something actionable.

## What I've Learned

Building this has been the best hands-on education in applied AI development I could have gotten. Working with Claude Code and Cursor daily has fundamentally changed how I think about what a PM with technical fluency can ship independently. The gap between idea and working product is smaller than it's ever been if you know how to work with these tools.

## Tech Stack

React, Supabase, Vercel, Claude Code, Cursor, Lovable, real-time weather and resort condition APIs, travel pricing integrations
